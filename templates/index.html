📖 如何获取 Cookie？

  1. 用 Chrome 打开 weread.qq.com 并扫码登录
  2. 按 F12（Mac：⌘⌥I）打开开发者工具
  3. 点击顶部 Network（网络） 标签
  4. 刷新页面，在请求列表中随便点一个
  5. 右键 → Copy → Copy as cURL
  6. 粘贴到左侧输入框即可

💡 也可直接复制右键菜单中 Headers → cookie 的值（纯字符串格式）。

⚠️ Cookie 有效期约数天，过期后需重新获取。勾选「记住」可在刷新后自动填充。
